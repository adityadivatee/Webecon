## 🛠️ Game Engine Integration

### **Unity Setup**
1. Copy the `Unity_Webecon` folder into your `Assets/` directory.
2. Use the `Webecon.Draw()` function in your code to render any icon dynamically.
3. High-performance JSON-based data loading ensures zero lag in your Scene.

### **Unreal Engine Setup**
1. Integrate the `Source/Public` headers into your project.
2. Link the `Content/Icons` folder to your project directory.
3. Access icons natively via C++ or Blueprints.
