# Webecon Dart SDK (v17.2.7 PRO)

## 🚀 Installation
Include `webecon.dart` in your project's source directory.

## 🛠️ Quick Usage
```dart
var tag = Webecon.icon("zap").setSize(48).setTheme("neon").build();
```
