# Webecon Go SDK (v17.2.6 PRO)

## 🚀 Installation
Include `webecon.go` in your project's source directory.

## 🛠️ Quick Usage
```go
import "webecon"
tag := webecon.Icon("zap").SetSize(48).SetTheme("neon").Build()
```
