# Webecon Go SDK (v17.2.8 PRO)

> ⚠️ **Public Beta**: Webecon is currently in Public Beta. Please report any bugs or issues to [support@adityadivte.com](mailto:support@adityadivte.com).

## 🚀 Installation
Include `webecon.go` in your project's source directory, or install via package manager if applicable.

## 🛠️ Quick Usage
```go
import "webecon"
tag := webecon.Icon("zap").SetSize(48).SetTheme("neon").Build()
```
