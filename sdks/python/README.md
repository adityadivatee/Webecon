# Webecon Python SDK (v17.2.5 PRO)

## 🚀 Installation
Include `webecon.py` in your project's source directory.

## 🛠️ Quick Usage
```python
import webecon
icon = webecon.Webecon("zap").size(48).theme("neon").trigger("hover")
print(icon)
```
