## 🚀 Installation
Include `webecon.py` in your project's source directory, or install via package manager if applicable.

## 🛠️ Quick Usage
```python
import webecon
icon = webecon.Webecon("zap").size(48).theme("neon").trigger("hover")
print(icon)
```
