# Webecon Java SDK (v17.2.6 PRO)

## 🚀 Installation
Include `com\adityadivatee\Webecon\Webecon.java` in your project's source directory.

```java

String svg = Webecon.icon("home").size(48).theme("neon").trigger("hover").build();
```
