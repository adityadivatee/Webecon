# Webecon — Universal Animated Icon Engine

![Webecon Logo](https://raw.githubusercontent.com/adityadivatee/Webecon/main/Webecon.png)

**1,031+ Premium Animated Icons | Native SVG Rendering | 100% Free for Commercial Use**

Webecon is a professional-grade iconography ecosystem designed for high-performance Flutter applications. It provides a unified API for 1,031+ themeable, animated icons that work natively on Android, iOS, and Web.

## 🌟 Key Features

- **🦾 Native Performance**: Uses `flutter_svg` for crisp, hardware-accelerated rendering.
* **✨ Theme Support**: Switch between 'line', 'shine', 'neon', and more.
* **🎬 Motion Presets**: Optimized for smooth animations.
* **🎨 Dual-Color System**: Deep branding integration with primary and secondary colors.

## 🛠️ Installation

Add `webecon` to your `pubspec.yaml`:

```yaml
dependencies:
  webecon: ^17.2.6
```

## 🚀 Usage

```dart
import 'package:webecon/webecon.dart';

// The Standard Way
Webecon(
  name: 'rocket',
  size: 64,
  color: Colors.blue,
  theme: 'shine',
)

// The Fluent Way
Webecon.icon('zap', size: 48, color: Colors.amber)
```
