# Webecon — Universal Animated Icon Engine (NPM)

![Webecon Banner](https://raw.githubusercontent.com/adityadivatee/Webecon/main/Webecon.png)

**1,031+ Premium Animated Icons | Native Web Components | React Support | 100% Free**

Webecon is a professional iconography ecosystem for the modern web. This package provides everything you need to use animated icons in any framework (React, Vue, Angular) or Vanilla JS.

## 🚀 Installation

```bash
npm install webecon
```

## 🛠️ Usage

### 1. In Vanilla JS / HTML
Just import the package and use the custom element.

```javascript
import 'webecon';
```

```html
<webecon-icon name="rocket" size="64" color="#6366f1" theme="shine"></webecon-icon>
```

### 2. In React
We provide a first-class React component.

```jsx
import { Webecon } from 'webecon/react';

function App() {
  return (
    <Webecon name="heart" size={48} color="red" theme="shine" />
  );
}
```

## ✨ Supported Themes
- `line` (Classic)
- `shine` (Glassmorphism)
- `neon` (Glow)
- `duo` (Dual-color)
- `glass` (Transparent)
