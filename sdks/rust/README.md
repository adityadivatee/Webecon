# Webecon Rust SDK (v17.2.6 PRO)

## 🚀 Installation
Include `webecon.rs` in your project's source directory.

## 🛠️ Quick Usage
```rust
use webecon::Webecon;
let tag = Webecon::icon("zap").size(48).theme("neon").build();
```
